# MyNotetion

## Technologies

|                     |
|---------------------|
| Angular 14          |
| TypeScript          |
| Bootstrap 5.3       |
| Auth (LocalStorage) |

## Installation and Setup

- Setup environment

```
yarn install
```

- Development server

```
ng serve
```

- Build

```
ng build
```

- Unit tests

```
ng test
```

- End-to-end tests

```
ng e2e
```
